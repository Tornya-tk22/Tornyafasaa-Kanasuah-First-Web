General Idea Web on Development

Web development is the method of creating websites where you will have to create many webpages and link all at once. Web development is basically divided into two parts:

Website development is the creation of webpages which are combined together to make a website. Here, we will learn ho to create a website using two programming languages which are used mostly for beginners. They are: HTML & CSS.

    HTML (HyperText Markup Language) is use to structure content on a webpage, it is the standard language for creating web pages. A CSS file must be linked to a html file using the link element [link] in order for it to have your css file effective on your website in the html file. In order to start a html file, your computer must be able to understand the program that it is about to read. You must always save a html file with the extension [.html] this will tell the computer that it is a html file.Once the file is save as html file, it will take the form of your default browser. Always save your project before running it in your browser, and if your browser is already open in your work, save your project and then refresh your browser.


    HTML in a simple term, it's the skeleton of the webpage. Here's a basic structure of an HTML document:

    Explaination

    • "!DOCTYPEhtml": Declares the document type and version of HTML.

    • "html lang="en": this interprates to the what type of language the program is written in.

    • "body": this contain the information that you want to be visible to the viewers of your website.

    • "head": this contain the meta information of the website that the browser understands and interprates (eg: chatset, viewport, title......)

    HTML has elements that are used to layout your website. These elements are grouped in two categories and they are:

  • Block Element: are elements that take up the whole width of a page regardless of the content. Eg. 'div', 'p', 'h2', 'ul', 'li', etc.
  • Inline Element: are elements that donâ€™t take up more space than their content needs. They are line up side by side in the browser. Eg. 'span', 'img', 'strong', 'em' etc

    • All elements in html has an openning and clossing tag. All tags must be closed in order to advoid inproper coding. Here are some elements and their function in an html file.

    • 'p' : this is used to input paragraph on your website.

    • 'h'{h1, h2,......h6}: this element is used to write a heading in your website.

    • 'li': this element customizes the list of items in your webpage

    • 'ul': it specifies that the list of items are in an unordered list, which means that it's not listed with numbers but points or dots.

    • 'ol': it specifies that the list of items are in an ordered list, which means that it's listed with numbers.

    • 'img': this' used to input photos on your website. This element comes with an (src=" ") tag. This 'src' stands for source, therefore it's the photo name that will be place in the 'src' tag within the 'img' element along with the extension of the photo {.jpg, .png, etc}


    • 'video': this' used to install videos on your website. This element also comes with an (src=" ") tag. Therefore it's the video's name that will be place in the 'src' tag within the 'img' element along with the extension of the video {.mp4}

    CSS Basics




    CSS (Cascading Style Sheet) is use to design the layout of a webpage. A CSS file must be linked to a html file in order for it to have your css file effective on your website in the html file. In order to start a css file, your computer must be able to understand the program that it is about to read. You must always save a css file with the extension [.css] this will tell the computer that it is a css file. Once the file is save as css file, it will take the form of your css file manager on your pc. Always save your project before running it in your browser, and if your browser is already open in your work, save your project and then refresh your browser.


    CSS Syntax

    A CSS rule-set consists of a selector, curly braces, and a declaration block:

    • Selectors: they are used to target an element on a webpage.
    • Declaration: is a set of key value pair.


    Selectors

    • Element Selector: this selects all elements of the given type. [p, h2, div]
    • Class Selector: this selects all elements of the given class. A class can be installed in the head of an element. [e.g. ("div class= ' myClass'")]. A class can be selected in a css with a dot attached to it. [.myClass]
    • ID Selector: Selects a single element with a specific ID attribute. An ID in a html file can appear like this [header id="myID"] and in the css file, can be selected like this [#myID]. An ID can only be attached to a specific element.
    • Attribute Selector: Selects elements with a specific attribute.

    CSS has many declarations that are used to design elements in your html file. Some of those elements are:

    background-color, color, text-align, display, width, height, border, font design(font-weight, font-style, font-family.........)

    HTML and CSS Code Example

    HTML File (index.html)

    
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content=" 
     width=device-width, initial-scale=1.0">
                <title>Simple Web Page</title>
                <link rel="stylesheet" 
     href="styles.css">
            </head>
            <body>
                <h1>Welcome to My Web Page</h1>
                <p>This is a simple example of an 
     HTML and CSS file for beginners.</p>
                <button class="styled-button"> 
     Click Me!</button>
            </body>
            </html>
